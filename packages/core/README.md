# The RxStack Core

> `@rxstack/core` is set of components which are the backbone of rxstack framework.

## Installation

```
npm install @rxstack/core --save

// peerDependencies
npm install @rxstack/async-event-dispatcher@^0.5 @rxstack/exceptions@^0.5 winston@^3.2.1
```
